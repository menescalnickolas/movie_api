Movie API Documentation

Introduction

Welcome to the User Management API! This API allows developers to manage a collection of users with simple CRUD operations. The API provides endpoints for creating, reading, updating, and deleting user records. Each user has a unique ID, a name, and a list of favorite movies.

Features

  • Create a New User: Add a new user to the database.
  • Retrieve Users: Get a list of all users or a specific user by ID.
  • Update User Information: Modify the details of an existing user.
  • Delete a User: Remove a user from the database.
