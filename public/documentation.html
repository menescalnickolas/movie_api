Introduction

Welcome to the User Management API! This API allows developers to manage a collection of users with simple CRUD operations. The API provides endpoints for creating, reading, updating, and deleting user records. Each user has a unique ID, a name, and a list of favorite movies.

Features

  • Create a New User: Add a new user to the database.
  • Retrieve Users: Get a list of all users or a specific user by ID.
  • Update User Information: Modify the details of an existing user.
  • Delete a User: Remove a user from the database.
Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Create New User /users POST None A JSON object with the user's information or "Users need names".
Update User Info /users/:id PUT A JSON object with the user's updated information. For example: {"name": Updated Name} A JSON object with the user's updated information or "No user found.".
Add Movie to List of Favorites /users/:id/:movieTitle POST None A message saying "${movieTitle} has been added to user ${id}'s array." OR "No user found.".
Remove Movie to List of Favorites /users/:id/:movieTitle DELETE None A message saying "${movieTitle} has been removed from user ${id}'s array." OR "No user found.".
Allow Users to De-Register /users/:id DELETE None A message saying "User ${id} has been deleted." OR "No user found.".
Get All Movies /movies GET None A JSON object with data of all the movies.
Get Data Of A Single Movie By Title /movies/:title GET None A JSON object with data about the movie OR the message "Movie not found!".
Get Data Of A Genre By Genre Name /movies/genre/:genreName GET None A JSON object with data about the genre OR the message "Genre not found!".
Get Data Of A Director By Their Name /movies/directors/:directorName GET None A JSON object with data about the Director OR the message "Director not found!".
